
# Vakinha dos Bastiões – React + Vite

Site solidário inspirado no Vakinha.com.br, pronto para deploy na Vercel.

## Deploy na Vercel

1. Faça login na Vercel (https://vercel.com/) com sua conta.
2. Clique em "New Project" e conecte o repositório deste projeto (ou faça upload manual).
3. Escolha o framework "Vite" (detectado automaticamente).
4. Clique em "Deploy".

### Dicas
- O arquivo `public/vakinha-banner.jpg` é um placeholder. Substitua por uma imagem real.
- O site é 100% responsivo e pronto para mobile.
- Para rodar localmente: `npm run dev`
